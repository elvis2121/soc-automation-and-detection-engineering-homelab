# Security Policy

## Supported use

This repository is a defensive homelab write-up and GitHub Pages publication set. It is intended for documentation, learning, and portfolio review.

## Reporting a concern

If you notice:

- exposed secrets,
- unsafe sample data,
- misleading security claims, or
- content that could create accidental risk,

please open a private security report through GitHub's security reporting flow if available, or contact the repository owner directly before opening a public issue.

## Out of scope

- Requests to operationalize offensive techniques against third-party systems
- Reports based on attacking GitHub Pages hosting for this documentation repo
- Requests to recover hidden or removed credentials from screenshots
